Vuindows
======

Windows-like web OS made with Vue. Made for the game [Unexpected End](https://store.steampowered.com/app/740170/Unexpected_End/)

Setup
------

`npm install`

`npm run dev`

###Hot reload

`npm run hot`

Go to [http://localhost:8080]()
